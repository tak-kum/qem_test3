RL78/G24 Motor Control Evaluation Kit

RTK0EMG24SS00000BJ

The RL78/G24 Motor Control Evaluation Kit is a development kit that enables easy evaluation of motor control using permanent magnet synchronous motors (brushless DC motors). With this product and the sample code that can be downloaded from the website, you can start evaluating motor control using the RL78/G24 right away.

Product Specification

    RL78/G24 Motor Control Evaluation Kit
    • The CPU card is equipped with the 16-bit microcontroller RL78/G24 as the motor control MCU.
    • The permanent magnet synchronous motors (brushless DC motors) can be controled by hall sensors, encoder or sensorless control.
    • Supports 1-shunt/2-shunt/3-shunt current sensing
    • Supports zero-cross detection using on-chip comparator.
    • Supports the motor control development support tool : Renesas Motor Workbench .
    • Supports the development assistance tool for motor applications : QE for Motor .
    • By replacing the included CPU card with other CPU card, motor control with other MCU can be evaluated.
    • Included Kit Hardware
      • RL78/G24 (R7F101GLG2DFB) CPU card (RTK0EMG240C00000BJ)
      • Inverter board (RTK0EMGPLVB00000BJ)
      • Communication board (MC-COM)
      • Permanent magnet synchronous motor (TG-55L-KA)
      • Accessories (cables, standoffs, etc.)
    • Target Devices
      • MCU : R7F101GLG2DFB
      • Gate Driver : HIP2101IBZ
      • MOSFET : RJK0703DPP-A0
      • Regulator : ISL8560IRZ,ISL85003FRZ
      • Operational amplifier : READ2302GSP
